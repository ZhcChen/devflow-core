# 待开发清单（方案已定稿）

用于记录已完成 PRD 与技术方案定稿、等待进入开发的任务。

## 记录规则

- 仅收录已完成“方案定稿”的任务
- 每条记录需包含任务编号、名称、PRD/SPEC 路径与定稿日期
- 主 Agent 按清单顺序执行，进入开发后将其从清单移除
- 清单顺序需按逻辑依赖排序，确保后续执行正确
- 若发现依赖冲突，先调整清单顺序并同步备注
